Formula 1
=========

Formula 1 is a collection of compound components and frequently-used functions
that aims to make it easier to develop apps for Omniport. 

By pre-preparing these components and functions, Formula one makes it supremely
fast and extremely easy to develop apps that look great and integrate cohesively 
with the rest of the platform.
