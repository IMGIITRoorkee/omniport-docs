Clone scripts
=============

Cloning the numerous services and packages required by Omniport is not easy. Or,
should I say, *was* not easy. Now that cloning scripts have been written setting
up Omniport frontend is child's play.

Read on to learn how you can use these clone scripts to remove the effort from
setting up the codebase.
