Scripts
=======

Omniport comes with a number of Bash scripts to perform a number of actions.
From setting up the entire Omniport codebase to starting an app, everything that
can be scripted will be scripted. That's the Omniport way of things!

Read more about the scripts to know about their function, their customisation
flags and options.
